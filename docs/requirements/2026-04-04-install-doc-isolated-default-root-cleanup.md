# 需求冻结: 安装文档默认根目录清理

- 日期: 2026-04-04
- 任务入口: 用户同意继续清理安装文档，使其与隔离默认安装根实现保持一致

## 目标

把当前用户可见的安装文档与安装提示词中的默认安装根描述，从旧的原生宿主目录改为新的隔离目标根，同时保留“真实宿主配置文件仍由宿主本地管理”的边界说明。

## 范围

- `docs/install/**`
- `docs/install/prompts/**`
- 对应的 bundle 镜像安装文档

## 约束

- 不改历史审计 / 历史计划 / 旧 requirements 文档
- 不把真实宿主 `settings.json`、`opencode.json` 等本地配置面误写成隔离安装根
- 继续保留环境变量优先级说明

## 验收标准

- 安装文档中的默认安装根统一描述为 `*_HOME` 或 `~/.vibeskills/targets/<host>`
- `windsurf`、`openclaw`、`opencode` 的宿主专页、推荐安装页、手动安装页、安装规则、宿主策略和提示词全部对齐
- `claude-code` / `cursor` / `codex` 保留对真实宿主 settings 文件的本地管理说明
- bundle 镜像与根文档保持一致

## 非目标

- 不做全仓历史文档迁移
- 不修改实现代码
